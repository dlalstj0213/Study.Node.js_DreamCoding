# Node.js

### ***"JavaScript Everywhere"***

- Node.js는 자바스크립트 런타임 환경이고 크롬의 V8 엔진을 이용하고 있다.
- Node.js는 오픈소스, 크로스플랫폼, 백엔드에서 사용되는, 즉, 브라우저 밖에서도 작동할 수 있는 자바스크립트 런타임 환경이다.

### *"Node.js is*
- ***JavaScript Runtime"***
- ***Single Tread"***
- ***Non-Blocking I/O"***
- ***Event-Driven"***
